# Happy 5 Months ❤️

A beautiful, interactive anniversary website made with love.

## Features

✨ **Passcode Protected** - Unlock with a special date
🎵 **Music Player** - Background music with floating player
💌 **Love Letter** - Typewriter effect letter with smooth animations
🌸 **Falling Petals** - Beautiful petal animations
❤️ **Interactive Hearts** - Click anywhere to create heart effects
✨ **Sparkle Effects** - Mouse movement creates sparkles
📱 **Fully Responsive** - Works perfectly on all devices

## How to Use

1. Replace `images/photo.jpg` with your special photo
2. Replace `music/song.mp3` with your favorite song
3. Update the passcode in `script.js` (currently set to "100226")
4. Customize all text in `index.html` to your liking

## Passcode

The default passcode is `100226` (February 10, 2026 - your special date). Change this in `script.js` line 65.

## Customization

Feel free to:
- Change colors in `style.css`
- Update all personal messages in `index.html`
- Adjust animation speeds and effects
- Add more sections as needed
